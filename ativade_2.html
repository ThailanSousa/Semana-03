1. O que o algoritmo faz?

Este algoritmo foi desenvolvido para calcular o centro de massa de um conjunto de dados sem utilizar funções de alto nível como .mean() ou sum(). Ele opera através de iteração direta, garantindo controle total sobre o processo de soma e contagem.

Fluxo Lógico

Coleta:

Extrai os dados brutos da coluna do DataFrame.

Acúmulo:

Percorre a lista somando cada valor individualmente.

Divisão:

Calcula o quociente entre a soma e a quantidade total.

3. Código da Função

def calcular_media(df, coluna):
    # Converte para lista comum
    lista_valores = df[coluna].tolist()
    
    soma_total = 0
    contagem = 0
    
    for valor in lista_valores:
        soma_total += valor
        contagem += 1
    
    media = soma_total / contagem
    return media

Insight Analítico

A média de 8.0 funciona como o centro de gravidade das notas. Note que o "8" da Ana está exatamente na média, enquanto o "6" do Bob está 2 pontos abaixo e o "10" da Cris está 2 pontos acima. Isso demonstra a propriedade da média de equilibrar as dispersões. Em análise de dados, isso nos dá uma visão rápida da performance, mas exige cuidado com valores extremos (outliers).
