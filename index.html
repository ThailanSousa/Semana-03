Desenvolvimento de um algoritmo manual para cálculo de tendências centrais, aplicado em cenários reais de atendimento ao cliente e performance de marketing.

calcular_media.py 
def calcular_media(df, coluna):
    # Extraímos os dados para uma lista simples
    lista_valores = df[coluna].tolist()
    
    # Inicialização do Acumulador e Contador
    soma_total = 0
    contagem = 0
    
    for valor in lista_valores:
        soma_total += valor
        contagem += 1
    
    # Cálculo final (Divisão Aritmética)
    media = soma_total / contagem
    return media

O que faz?

Simula o comportamento de bibliotecas de alto nível usando um loop para somar e contar registros sem funções internas.

Visibilidade

Código formatado com alto contraste para garantir leitura clara em qualquer dispositivo ou apresentação.

Fundamento Matemático

x̄ = Σx / n

x̄: Resultado da Média

Σx: Somatório Total dos Valores

n: Quantidade Total de Amostras

Passo a Passo

  • 1 Extração dos dados brutos do DataFrame.
  • 2 Varredura (Loop) individual de cada linha.
  • 3 Acúmulo de valores e incremento do contador.
